整机程序：
实现功能：按通讯协议实现串口控制电路板进行各种特性参数的测量

在主函数中进行指令的识别和相应操作的执行

恒流源：SW高电平时输出大电流：2.5*10^(-4)A 
              SW低电平时输出小电流：0.5*10^(-6)A
暗电阻测量：片外ADS1211 恒流源小电流档位
亮电阻测量：片外ADS1211 恒流源大电流档位
LED电流设置：片内ADC输出电压转换为电流 换算公式 I=DHR/12409(A)
时间响应特性测量：片内AD连续转换模式 定时器2中断每10ns给出信号 控制AD获取采样结果
